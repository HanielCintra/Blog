Code samples for
http://tpodolak.com/blog/2015/04/12/using-ecmascript6-today/

Before launching index.html run in commandline

```bash
npm install gulp -g
npm install
```
and then
```bash
gulp --traceur // to transpile code using traceur
```
or
```bash
gulp --babel // to transpile code using babel
```
